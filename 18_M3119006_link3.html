Membuat Link dengan Situs Tertentu

A. Pendahuluan :

Fungsi dari link adalah menghubungkan satu halamn web ke halaman web lain. Caranya dengan mengklik suatu obyek berupa teks maupun gambar. Untuk membuat link dengan memberikan tag < a href="url" > dan diakhiri dengan < /a >.

B. Script Code

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title> Link </title> </head> <body> <h1> Link </h1> <p> Klik link berikut <a href="http://www.google.com"> Google </a> </p> </body> </html>

C. Hasil Preview

Link

Klik link berikut Google

D. Kesimpulan :

Dari Script dan hasil di atas dapat diketahui bahwa dengan membuat link yang ditargetkan dengan link suatu situs web tertentu maka muncul tulisan alamat website pada link ketika kursor didekatkan ke arah link tersebut. Sehingga ketika di klik maka langsung diarahkan ke website yang dituju.
