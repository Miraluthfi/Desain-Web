Monospaced Font

A. Pendahuluan :

Monospaced Font digunakan jika ingin mempublish code yang digunakan dalam web tersebut. Untuk membuat monospaced font tersebut menggunakan tag < code > dan diakhiri dengan < /code >.

B. Script Code

<!DOCTYPE> <html> <head> <title> Monospace </title> <h1 style ="text-align:center;"><u> Monospace Font </u></h1> </head> <body> <code> &lt;?php<br /> $mangga = &quot;manalagi&quot;;<br /> $lampu = &quot;phillips&quot;;<br /> $hewan = &quot;sapi&quot;;<br /> $kuliah = &quot;d3ti&quot;;<br /> $macamobjek = $mangga . $lampu . $hewan . $kuliah;<br /> echo $macamobjek;<br /> ?&gt; </code> </body> </html>

C. Hasil Preview

<?php
$mangga = "manalagi";
$lampu = "phillips";
$hewan = "sapi";
$kuliah = "d3ti";
$macamobjek = $mangga . $lampu . $hewan . $kuliah;
echo $macamobjek;
?>

D. Kesimpulan :

Dari Script dan hasil di atas dapat diketahui bahwa terlihat sebuah code script PHP yang digunakan unutk membuat web.
