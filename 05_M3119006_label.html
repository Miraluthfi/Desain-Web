Cara Membuat Label pada HTML

A. Pendahuluan :

Sebuah elemen pada HTML diberi label yang akan memiliki efek muncul tool tip pada browser ketika kursor didekatkan pada elemen tersebut. Untuk memberikan label pada elemen, dengan menambahkan title="label" pada tag pembuka elemen.

B. Script Code

<!DOCTYPE> <html> <head> <title> Label </title> <h1 style ="text-align:center;"><u> Cara Membuat Label pada HTML </u></h1> </head> <body> <h1 title=" World Wide Web ">WWW</h1> <p> <h1 title="Short Message Service">SMS</h1> </p> </body> </html>

C. Hasil Preview

WWW

SMS

D. Kesimpulan :

Dari Script dan hasil di atas dapat diketahui bahwa bilamana cursor didekatkan pada tulisan WWW atau elemen tersebut maka akan muncul tool tip seperti kotak kecil/ label bertuliskan kepanjangan dari WWW tersebut. Bilamana tag title diletakkan dalam paragraf seperti hasil preview di atas pada tulisan SMS, maka akan muncul juga tool tip seperti kotak kecil/ label bertuliskan kepanjangan dari SMS tersebut.
