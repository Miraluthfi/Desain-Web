Cara Membuat List Box pada HTML

A. Pendahuluan :

Form dalam desain web merupakan hal vital yang mutlak harus ada untuk membuat keinteraktifan dengan pengunjung. Berikut salah satu komponen pada form ialah List Box, list box pada prinsipnya sama dengan combo box, hanya tidak terdapat drop down button. Semua item pilihan langsung ditampilkan dalam box sekaligus. Perintahnya sama dengan combo box, hanya perbedaannya ialah dengan adanya atribut size="n" pada tag < select name="nama komponen">, dengan n adalah jumlah item yang akan ditampilkan.

B. Script Code

<!DOCTYPE> <html> <head> <title> Membuat List Box </title> <h1 style ="text-align:center;"><u> Cara Membuat List Box pada HTML </u></h1> </head> <body> <h1> Pekerjaan Anda </h1> <form method="post" action=""> <select name="select" size="4"> <option value="dosen"> Dosen </option> <option value="dokter"> Dokter </option> <option value="petani"> Petani </option> <option value="karyawan"> Karyawan </option> </select> <input type="submit" name="submit" value="Kirim data" /> </form> </body> </html>
