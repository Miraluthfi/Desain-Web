Membuat Link dalam Satu Folder yang Sama dengan Induknya

A. Pendahuluan :

Fungsi dari link adalah menghubungkan satu halaman web ke halaman web lain. Caranya dengan mengklik suatu obyek berupa teks maupun gambar. Untuk membuat link dengan memberikan tag < a href="url" > dan diakhiri dengan < /a >.

B. Script Code

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title> Link </title> </head> <body> <h1> Link </h1> <p> Klik link berikut <a href="page2.html"> Link </a> </p> </body> </html>

C. Hasil Preview

Link

Klik link berikut Link

D. Kesimpulan :

Dari Script dan hasil di atas dapat diketahui bahwa dengan membuat link yang ditargetkan ke HTML yang telah saya buat dan tersimpan si sebuah folder yang sama. Sehingga ketika link di klik maka muncul file folder website yang telah saya buat tadi. Ketika kursor didekatkan link tersebut makan kursor berubah bentuk jari telunjuk dan bagian bawah pojok kiri terlihat kotak kecil dari penyimpanan link tersebut. Sehingga ketika di klik maka langsung diarahkan ke file tersebut.
