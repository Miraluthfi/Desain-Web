Cara Membuat Login pada HTML

A. Pendahuluan :

Form dalam desain web merupakan hal vital yang mutlak harus ada untuk membuat keinteraktifan dengan pengunjung. Berikut salah satu komponen pada form ialah Text Box, dengan membuat text box ini khusus digunakan untuk memasukkan data rahasia, misalnya password. Text box jenis ini sering digunakan untuk keperluan login karena data password login yang dimasukkan ke text box tidak dapat dibaca dengan jelas. Hal ini dikarenakan data text ditampilkan dalam bentuk simbol tertentu. Untuk membuat text box login, caranya dengan menuliskan < input type="text" name="nama textbox" value="nama komponen" >

B. Script Code

<!DOCTYPE> <html> <head> <title> Membuat Login </title> <h1 style ="text-align:center;"><u> Cara Membuat Login pada HTML </u></h1> </head> <body> <form method="post" action="form.hmtl"> First name: <br/> <input type="text" name="firstname" value="Mickey"> <br/> <br/> <input type="submit" value="submit"> </form> <p> Jika anda submit data melalui tombol submit maka data akan ditampilkan pada halaman "form.html". </p> </body> </html>
