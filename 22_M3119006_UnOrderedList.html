Membuat Unordered List pada HTML

A. Pendahuluan :

Unordered list merupakan list yang tidak terurut, biasanya berupa bulleted list (list dengan simbol tertentu). Untuk membuat unordered list < ul >, dengan memberikan tag < ul > dan diakhiri dengan < /ul >. Sedangkan daftar Item ditulis di antara tag < ul > dan < /ul >, dengan diapit tag < li > dan < /li >.

B. Script Code

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title> Unordered List </title> </head> <body> <u1> <li> Jeruk </li> <li> Apel </li> <li> Duku </li> <li> Stroberi </li> <li> Nanas </li> </u1> </body> </html>

C. Hasil Preview

  • Jeruk
  • Apel
  • Duku
  • Stroberi
  • Nanas
    •

    D. Kesimpulan :

    Dari Script dan hasil di atas dapat diketahui bahwa terdapat beberapa list terhadap suatu itemnya dengan bulleted list.
