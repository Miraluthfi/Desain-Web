Preformatted Text

A. Pendahuluan :

Preformatted Text mempunyai fungsi yang sama dengan Monospaced Font. Namun bedanya Preformatted text lebih memudahkan jika terdiri dari baris yang banyak. Untuk membuat preformatted text dengan memberikan tag < pre > dan diakhiri dengan < /pre >.

B. Script Code

<!DOCTYPE> <html> <head> <title> Performatted Text </title> <h1 style ="text-align:center;"><u> Preformatted Text </u></h1> </head> <body> <pre> &lt;?php<br /> $mangga = &quot;manalagi&quot;;<br /> $lampu = &quot;phillips&quot;;<br /> $hewan = &quot;sapi&quot;;<br /> $kuliah = &quot;d3ti&quot;;<br /> $macamobjek = $mangga . $lampu . $hewan . $kuliah;<br /> echo $macamobjek;<br /> ?&gt; </pre> </body> </html>

C. Hasil Preview 

	<?php

	$mangga = "manalagi";

	$lampu = "phillips";

	$hewan = "sapi";

	$kuliah = "d3ti";

	$macamobjek = $mangga . $lampu . $hewan . $kuliah;

	echo $macamobjek;

	?>

D. Kesimpulan :

Dari Script dan hasil di atas dapat diketahui bahwa terlihat sebuah code script PHP yang digunakan unutk membuat web sama seperti monospace text, namun tampilan pada preformatted text ini spacenya lebih berjarak dan lebih menjorok ke arah kanan daripada dengan monospace yang spacenya tidak ada jaraknya.
