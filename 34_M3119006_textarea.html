Cara Membuat Text Area pada HTML

A. Pendahuluan :

Form dalam desain web merupakan hal vital yang mutlak harus ada untuk membuat keinteraktifan dengan pengunjung. Berikut salah satu komponen pada form ialah Text Area, Komponen text area mirip dengan text box. Perbedaanya adalah bahwa text area, perintahnya : < textarea name="nama komponen"> < /textarea > dengan menambahkan atribut rows="n" dan cols="n" dengan n adalah banyaknya baris serta kolom untuk mengatur text panjang baris dan kolom tertentu.

B. Script Code

<!DOCTYPE> <html> <head> <title> Membuat Text Area </title> <h1 style ="text-align:center;"><u> Cara Membuat Text Area pada HTML </u></h1> </head> <body> <h1> Alamat Anda </h1> <form method="post" action=""> <textarea name="alamat" rows="S" cols="40"></textarea> <br /> <br /> <input type="submit" name="submit" value="Kirim data" /> </form> </body> </html>
