Membuat Favicon pada Website

A. Pendahuluan :

Favicon (favorite icon) adalah suatu image (icon) yang akan tampil pada address bar, menu favorites pad abrowser. Penempatan perintah membuat favicon terletak di dalam blok header.

B. Script Code

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title> M3119006_Error Page </title> <link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png"> </head> <body> <h1> Error </h1> <p> This is an error page. This is an error page. This is an error page. This is an error page. </p> </body> </html>
