Membuat ShortcutLink pada HTML

A. Pendahuluan :

Dengan adanya shortcut link, maka pengunjung web dapat membuka link meskipun tanpa menggunakan mouse. Untuk membuat shortcutlink caranya dengan menambahkan atribut accesskey="shortcut" pada < a href="url" >. Dengan catatan sebaiknya hindari huruf shortcut yang sama dengan shortcut menu pada aplikasi browser.

B. Script Code

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title> ShortcutLink </title> </head> <body> <a href="#hal1" accesskey="v"> Halaman 1 </a> (Alt+V, Ctr+V) <br /> <a href="#hal2" accesskey="v"> Halaman 2 </a> (Alt+W, Ctr+W) <br /> <a href="#hal3" accesskey="v"> Halaman 3 </a> (Alt+X, Ctr+X) <br /> </body> </html>

C. Hasil Preview

Halaman 1 (Alt+V, Ctr+V)
Halaman 2 (Alt+W, Ctr+W)
Halaman 3 (Alt+X, Ctr+X)

D. Kesimpulan :

Dari Script dan hasil di atas dapat diketahui bahwa pengunjung web dapat membuka link tanpa menggunakan mouse. Konsep ini mengadopsi shortcut menu pada aplikasi berbasis GUI.
