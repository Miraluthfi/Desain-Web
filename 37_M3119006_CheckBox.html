Cara Membuat Check Box pada HTML

A. Pendahuluan :

Form dalam desain web merupakan hal vital yang mutlak harus ada untuk membuat keinteraktifan dengan pengunjung. Berikut salah satu komponen pada form ialah Check Box, seperti halnya radio button, check box juga digunakan untuk input yang berupa pilihan. Penampilannya pun hampir sama dengan radio button, namun perbedaanya bahwa dalam check box memungkinkan user memilih item lebih dari satu. Tag untuk membuat check box hampir sama dengan radio button yaitu < input type="checkbox" name="nama komponen" value="nilai" />.

B. Script Code

<!DOCTYPE> <html> <head> <title> Membuat Check Box </title> <h1 style ="text-align:center;"><u> Cara Membuat Check Box pada HTML </u></h1> </head> <body> <h1> Pekerjaan Anda </h1> <form method="post" action=""> <input type="checkbox" name="hobi" value="memancing" /> Memancing <br /> <input type="checkbox" name="hobi" value="main game" /> Main Game <br /> <input type="checkbox" name="hobi" value="nonton film" /> Nonton Film <br /> <input type="checkbox" name="hobi" value="travelling" /> Travelling <br /> <input type="submit" name="submit" value="Kirim Data" /> </form> </body> </html>

D. Kesimpulan :

Dari Script dan hasil di atas dapat diketahui bahwa pemberian nama komponen pada check box sama seperti radio button. Apabila item pilihan check box masih dalam kelompok yang sama, maka digunakan nama komponen yang sama. Terlihat pada tampilan di atas terdapat pilihan item dan juga bisa memilih lebih dari satu item.
