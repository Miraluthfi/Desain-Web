Cara Membuat Valign pada HTML

A. Pendahuluan :

Atribut Valign dan Align dalam memformat cell digunakan untuk mengatur posisi data pada cell. Valign digunakan untuk mengatur posisi vertikal dan relatif terhadap cell, sedangkan Align mengatur posisi horizontal.

B. Script Code

<!DOCTYPE> <html> <head> <title> Membuat Valign </title> <h1 style ="text-align:center;"><u> Cara Membuat Valign pada HTML </u></h1> </head> <body> <H1> Harga Buah </H1> <table border="2" align="center" width="40%"> <tr> <td rowspan="3" valign="top"> Ini adalah Tabel </td> </tr> <tr> <td align="right"> HTML </td> </tr> <tr> <td> Perintah dan tahapan web dasar </td> </tr> </table> </body> </html>

C. Hasil Preview

Perintah dan tahapan web dasar

D. Kesimpulan :

Dari Script dan hasil di atas dapat diketahui bahwa tabel yang terbentuk seperti terlihat pada tampilan di atas.
