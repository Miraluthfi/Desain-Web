Mengubah Urutan List pada HTML

A. Pendahuluan :

Untuk ordered list, penomoran item tidak selalu dimulai dengan angka 1. Maka cara untuk menentukan angka awal penomoran item adalah dengan menambahkan atribut start="n" pada tag < ol > dengan n adalah angka awal penomoran yang diinginkan.

B. Script Code

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title> Urutan List </title> </head> <body> <h1> Cara Memasak Mie Instan </h1> <ol start="3"> <li> Masukkan 2 gelas belimbing air ke dalam panci </li> <li> Masukkan mie instan ke dalam panci</li> <li> Rebus mie instan selama kurang lebih 3 menit </li> <li> Masukkan bumbu dan minyak </li> <li> Siapkan mangkuk </li> <li> Tuangkan mie ke dalam mangkuk </li> </ol> </body> </html>

C. Hasil Preview

Cara Memasak Mie Instan

  1. Masukkan 2 gelas belimbing air ke dalam panci
  2. Masukkan mie instan ke dalam panci
  3. Rebus mie instan selama kurang lebih 3 menit
  4. Masukkan bumbu dan minyak
  5. Siapkan mangkuk
  6. Tuangkan mie ke dalam mangkuk

D. Kesimpulan :

Dari Script dan hasil di atas dapat diketahui bahwa dalam list tersebut dimulai dengan angka 3, karena kita menambahkan atribut start="n" pada tag < ol > dengan n = 3.
