Cara Mengupload File pada HTML

A. Pendahuluan :

Form dalam desain web merupakan hal vital yang mutlak harus ada untuk membuat keinteraktifan dengan pengunjung. Berikut salah satu komponen pada form ialah File Field, komponen file field sering digunakan untuk keperluan upload file ke server. Secara sekilas, penampilan komponen ini seperti text box namun terdapat tombol untuk melakukan browser file yang akan dipilih. Untuk membuat komponen ini, dengan memberikan tag < input type="file" name="nama komponen" />. Selain itu tambahkan atribut enctype="multipart/form-data" pada tag < form method="post" action="action.script" >.

B. Script Code

<!DOCTYPE> <html> <head> <title> Mengupload File </title> <h1 style ="text-align:center;"><u> Cara Mengupload File pada HTML </u></h1> </head> <body> <h1> Upload File </h1> <form action="" method="post" enctype="multipart/form-data"> <input type="file" name="fileku" /> <input type="submit" name="submit" value="Kirim Data" /> </form> </body> </html>
