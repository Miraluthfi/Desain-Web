Cara Membuat Heading pada HTML

A. Pendahuluan :

Tag Heading digunakan untuk membuat judul pada halaman web, tag heading pada HTML terdiri dari 6 tingkatan yaitu dari h1,h2,h3,h4,h5,h6. Semakin besar levelnya, maka ukuran font yang tampil akan semkain kecil. Untuk membuat heading menggunakan tag < hn > dan diakhiri < /hn >, dengan n = 1,2,3,4,5,6.

B. Script Code

<!DOCTYPE> <html> <head> <meta charset="UTF-8"> <title> Heading </title> <h1 style ="text-align:center;"> Heading </h1> <h1> Heading 1 </h1> <h2> Heading 2 </h2> <h3> Heading 3 </h3> <h4> Heading 4 </h4> <h5> Heading 5 </h5> <h6> Heading 6 </h6> <h7> Heading 7 </h7> <h8> Heading 8 </h8> <h9> Heading 9 </h9> <h10> Heading 10 </h10> </body> </html>

D. Kesimpulan :

Dari Script dan hasil di atas dapat diketahui bahwa semakin besar angka dalam tag maka hasilnya semakin kecil. Penggunaan heading hanya berlaku untuk h1 sampai h6, sehingga h7 sampai h10 tidak berlaku, maka tampilan tulisannya tetap seperti biasa.
