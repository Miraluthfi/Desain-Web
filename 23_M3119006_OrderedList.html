Membuat Ordered List pada HTML

A. Pendahuluan :

Ordered List merupakan list yang setiap itemnya terdapat huruf atau angka yang terurut. Biasanya Ordered list digunakan untuk membuat urutan langkah-langkah. Untuk memberikan ordered list dengan tag < ol > dan diakhiri dengan < /ol >. Sedangkan daftar Item ditulis di antara tag < ol > dan < /ol >, dengan diapit tag < li > dan < /li >.

B. Script Code

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title> Ordered List </title> </head> <body> <style> body{ background-image: url(full-bloom.png); } </style> <h1> Cara Memasak Mie Instan </h1> <o1> <li> Masukkan 2 gelas belimbing air ke dalam panci </li> <li> Masukkan mie instan ke dalam panci </li> <li> Rebus mie instan selama kurang lebih 3 menit </li> <li> Masukkan bumbu dan minyak </li> <li> Siapkan mangkuk </li> <li> Tuangkan mie ke dalam mangkuk </li> </o1> </body> </html>

C. Hasil Preview

Cara Memasak Mie Instan

  1. Masukkan 2 gelas belimbing air ke dalam panci
  2. Masukkan mie instan ke dalam panci
  3. Rebus mie instan selama kurang lebih 3 menit
  4. Masukkan bumbu dan minyak
  5. Siapkan mangkuk
  6. Tuangkan mie ke dalam mangkuk

D. Kesimpulan :

Dari Script dan hasil di atas dapat diketahui bahwa terdapat beberapa list terhadap suatu itemnya dengan ada angka secara berurutan seperti langkah-langkah.
